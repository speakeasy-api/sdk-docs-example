#### Authentication
To authenticate requests use the following authentication scheme with each request.

| Scheme                    | Field Name                | Location                  | Notes                     |
| ------------------------- | ------------------------- | ------------------------- | ------------------------- |
| API key                   | api_key                   | HTTP header               | `api_key: <YOUR_API_KEY>` |

#### Parameters


##### `headerParam1` _string_

##### `page` _integer_

##### `queryParam1` _string (optional)_
