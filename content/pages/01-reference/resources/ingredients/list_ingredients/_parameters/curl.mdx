#### Authentication
To authenticate requests use the following authentication scheme with each request.

| Scheme            | Field Name        | Location          | Notes             |
| ----------------- | ----------------- | ----------------- | ----------------- |
| API key           | Authorization     | HTTP header       | `Authorization: ` |

#### Parameters


##### `ingredients` _array (optional)_
A list of ingredients to filter by. If not provided all ingredients will be returned.
